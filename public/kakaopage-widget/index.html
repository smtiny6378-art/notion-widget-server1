카카오페이지 검색 → URL로 노션 저장
카카오페이지는 서버에서 직접 검색이 막혀서, 검색은 새 탭으로 열고
작품 URL을 붙여넣어 노션 DB에 저장하는 방식으로 동작해요.
1) 카카오페이지에서 검색 열기
웹툰/웹소설 모두 OK
새 탭에서 작품을 클릭해 작품 상세 페이지로 들어간 뒤, 주소창 URL을 복사해서 아래에 붙여넣어 주세요.
검색 결과 URL 예시는 https://page.kakao.com/search?keyword=무정의 처럼 생겼어요.
2) 작품 URL 붙여넣고 노션 저장
저장은 위젯에서
작품 ID: - 저장 API: 자동 탐지
⚠️ 브라우저 보안 때문에, 새 탭에서 열린 카카오페이지의 URL을 위젯이 “자동으로 가져올 수는 없어서” URL 붙여넣기 1번이 필요해요.
검색어를 입력하고 “검색 결과 열기”를 눌러 보세요.
