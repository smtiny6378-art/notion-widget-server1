카카오페이지 검색 → URL로 노션 저장
검색은 새 탭(카카오페이지)에서 하고,
작품 상세 URL을 붙여넣어 노션 DB에 저장해요.
1) 카카오페이지 검색 열기
웹툰/웹소설 모두 OK
✅ 검색 결과는 이 주소로 열려요: https://page.kakao.com/search/result?keyword=검색어
작품 URL 예시: https://page.kakao.com/content/65171279
2) 작품 URL 붙여넣고 저장
제목/표지 자동 채우기 지원
저장 시 서버가 작품 페이지의 메타(제목/표지/소개)를 최대한 가져와서 노션 속성에 채워요.
검색어를 입력하고 “검색 결과 열기”를 눌러 보세요.
