카카오페이지 → 노션 저장
✅ 추천 사용법: 카카오페이지 작품 상세에서 북마클릿(즐겨찾기) 클릭 → 자동 저장
또는 아래에 작품 URL을 붙여넣고 저장해도 돼요.
1) 카카오페이지에서 검색 열기
웹툰/웹소설 모두 OK
검색 결과 URL 예시: https://page.kakao.com/search?keyword=무정의
작품 URL 예시: https://page.kakao.com/content/12345678
2) 작품 URL 저장
자동 입력/자동 저장 지원
작품 ID: - 저장 API: 자동 탐지 자동 저장: -
북마클릿으로 들어오면 URL/제목이 자동으로 채워지고, auto=1이면 저장까지 자동으로 진행돼요.
검색어를 입력하고 “검색 결과 열기”를 눌러 보세요.
